CSS Box Model

Differences between margin, border, and padding

To start with, it's important to understand what the CSS Box Model is and how this is linked to HTML and CSS. The CSS Box Model is simply a box that wraps around an HTML element. The box is made up of seperate layers: padding, border, margin, and the centre of the box contains the HTML content. Every HTML element you create is wrapped around a box that can be manipulated using CSS.

  • Padding: Creates space around an HTML element (the centre of the box)

  • Border: The border goes around the HTML element and padding

  • Margin: Creates space around the border area - kind of like padding for the border

You can use length units such as PX (Pixels) to adjust padding and margin on all four sides of the box with the following CSS properties for padding: padding-top, padding-right, padding-bottom, padding-left, or you can simply use padding. You can use the following CSS properties for margin: margin-top, margin-right, margin-bottom, margin-left, or simply margin. This allows you to have more control over styling your HTML elements. The border thickness and style type can also be changed with length units and specific CSS properties.
